Which complaint categories have the most negative sentiment?

SELECT ai_category, COUNT(*) AS review_count, ROUND(100.0 * SUM(CASE WHEN ai_sentiment IN ('Negative', 'Very Negative') THEN 1 ELSE 0 END) / COUNT(*), 1) AS negative_pct, ROUND(AVG(CAST(star_rating AS FLOAT)), 2) AS avg_stars FROM classified_reviews GROUP BY ai_category ORDER BY negative_pct DESC;

Which product types generate the most critical / high-severity issues?

SELECT product_category, COUNT(*) AS total_reviews, SUM(CASE WHEN ai_severity = 'High' THEN 1 ELSE 0 END) AS high_severity, ROUND(100.0 * SUM(CASE WHEN ai_severity = 'High' THEN 1 ELSE 0 END) / COUNT(*), 1) AS high_severity_pct FROM classified_reviews GROUP BY product_category ORDER BY high_severity_pct DESC LIMIT 10;

Does AI sentiment align with star ratings? Cross-validation of model output.

SELECT star_rating, ai_sentiment, COUNT(*) AS count, ROUND(100.0 * COUNT(*) / SUM(COUNT(*)) OVER (PARTITION BY star_rating), 1) AS pct FROM classified_reviews GROUP BY star_rating, ai_sentiment ORDER BY star_rating, count DESC;

Are complaints growing or shrinking over time? Monthly negative review trend.

SELECT SUBSTR(review_date, 1, 7) AS month, COUNT(*) AS total, SUM(CASE WHEN ai_sentiment IN ('Negative', 'Very Negative') THEN 1 ELSE 0 END) AS negative_count FROM classified_reviews GROUP BY month ORDER BY month;

Executive risk summary with recommended escalation actions per category.

SELECT ai_category AS complaint_type, COUNT(*) AS volume, ROUND(AVG(CAST(star_rating AS FLOAT)), 1) AS avg_rating, SUM(CASE WHEN ai_severity = 'High' THEN 1 ELSE 0 END) AS high_risk_count, CASE WHEN SUM(CASE WHEN ai_severity = 'High' THEN 1 ELSE 0 END) > 50 THEN 'URGENT - escalate to product team' WHEN SUM(CASE WHEN ai_severity = 'High' THEN 1 ELSE 0 END) > 20 THEN 'MONITOR - flag for weekly review' ELSE 'OK - standard support queue' END AS recommended_action FROM classified_reviews GROUP BY ai_category ORDER BY high_risk_count DESC;
